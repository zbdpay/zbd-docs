---
title: "Get User Transactions"
description: "Retrieve a paginated transaction history for a widget user."
api: "GET https://api.zbdpay.com/api/v1/widget/users/{user_id}/transactions"
---

## Description

Returns a widget user's transaction history for your project. Use this from your server when you need to show reward funding and cashout activity outside the widget.

Amounts are returned in **currency minor units**. For USD, `100` equals `$1.00`.

## Configuration

### Header Parameters

<ParamField required header="apikey" type="string">
  Your ZBD project API key (production scope).
</ParamField>

### Path Parameters

<ParamField required path="user_id" type="string">
  The ZBD user ID returned by Create User.
</ParamField>

### Query Parameters

<ParamField query="skip" type="integer">
  Number of transactions to skip. Defaults to `0`.
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of transactions to return. Defaults to `20`; maximum is `100`.
</ParamField>

<ParamField query="type" type="string">
  Optional transaction type filter.
</ParamField>

<ParamField query="startDate" type="string">
  Optional ISO 8601 start date filter.
</ParamField>

<ParamField query="endDate" type="string">
  Optional ISO 8601 end date filter.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.zbdpay.com/api/v1/widget/users/4ac4fd8a-cc2c-4d03-af09-a76f4e89d652/transactions?skip=0&limit=20" \
  -H "apikey: YOUR_API_KEY"
```
</RequestExample>

## Package + Runtime Matrix

| Surface | Type | Runtime | NPM |
|---|---|---|---|
| `@axobot/cli` | CLI | Node.js >= 22 | Yes |
| `@axobot/fetch` | Library | Node.js >= 22 | Yes |
| `@axobot/pay` | Library | Node.js >= 22 | Yes |
| `agent-wallet-dashboard` | Next.js app | Node.js >= 22 | No |
| `zbd-ai` | Next.js app/service | Node.js >= 22 | No |

## Runnable Examples Coverage

| Repository | `examples/` directory | Fastest demo script |
|---|---|---|
| `axobot-fetch` | Yes | `examples/zbd-agent-fetch.mjs` |
| `axobot-pay` | Yes | `examples/http-server.mjs` |
| `axobot-cli` | No | Use `axo fetch` against `axobot-pay` example |
| `agent-wallet-dashboard` | No | Use `sample-data/` quick demo |
| `zbd-ai` | No | Use `POST /api/register` smoke test |

## Workspace Helpers

At the top-level agents workspace, these wrapper scripts forward to the sibling `agents-e2e` workspace for full-suite verification:

```bash
npm run install:all
npm run build:all
npm run test:all
npm run verify:all
```
